Expression Parameters

For parameters of type "Expression", values can be specified from various sources. Values for the expression parameters should be in one of the following format:

  • <Expression-type>:<Expression>
  • <Expression-type>(<java-type>):<Expression>

When format with java-type is used, the result of expression process, will be attempted to convert to specified java-type. If conversion fails an error will be thrown. Below list illustrates current expression types and their purpose.
